Producing a Differentiated Language-Learning Platform

Client: MyKstar

Term: Spring of 2023

Learn More

Project

overview

To develop a language learning platform starting with Korean as the initial language, we incorporated machine-learning handwriting recognition algorithms and karaoke features to attract investments for the platform by the latter half of 2023. We created an intuitive and enjoyable language platform, enhancing the language learning experience by personalizing user interactions.

Data Science

Leverage data, analytics, data visualization, and machine learning techniques to implement and enhance the experience on Project CUBS’ main deliverable

App Development

Create a differentiated language learning platform in mobile application

UI/UX Design

Design a language-learning app that can be easily used by users of different ages and genders.

Business Tech

Enhancing the language learning experience on the platform, whereby personalizing user experiences

Client

about the client

MyKstar is an up-and-rising startup focused on revolutionizing online virtual language learning. With a commitment to expanding and enriching the language learning experience. We are collaborating to incorporate innovative technology features such as karaoke and machine learning handwriting recognition features to enhance learners’ engagement and overall progress.

A Graduate-Student group team at the University of Columbia provided us with extensive research for enriching and improving our collaboration project. On a yearly basis, they specifically select a startup company to provide their consultation expertise. For this project, MyKstar was the selected client to collaborate with and provide the respective deliverables.

Machine-learning algorithm for teaching Korean on the online platform

Application Development Team

Previous Next

A differentiated language-learning platform for MyKstar

The aim of the application development team was to create a differentiated language learning platform. To develop key features such as Karaoke, a dictionary, and a word quiz. These features aimed to provide users with interactive and engaging tools to practice their language skills and enhance their learning experience.

  • Front-end: Designing an engaging interface with problem-solving activities, learning games, and notes on incorrect answers.
  • Back-end: Develop a centralized database for learning materials, user data management through login features, and an administrator server for customizing training materials.
  • Software Engineering and Game Development: Create language-learning games tailored to user needs, with selectable difficulty settings. Implement proper data management systems and saving features to ensure a seamless learning experience.

Business Technology Team

Data Analysis for enchaning user experience

The Business Technology team played a crucial role in achieving the objectives of Project CUBS, enhancing the language learning experience on the platform, whereby personalizing user experiences through the implementation of a secure CRM database for progress tracking, user-centric database interface, interactive dashboards, and an AI-powered chatbot.

  • Implement a secure and efficient backend CRM database that ensures data integrity and privacy while facilitating personalized user experiences and seamless progress tracking.
  • Design and visualize the database information in two customized formats to enhance usability and user satisfaction, catering to the distinct needs and preferences of a diverse group of platform users.
  • Create and deploy an AI-powered chatbot feature to provide round-the-clock user assistance, reducing response times, and optimizing operational efficiency to enhance the user experience.
